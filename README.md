# MIMO System over Rayleigh Fading Channel (MATLAB)

## Overview
This project focuses on the simulation and performance analysis of a MIMO communication
system operating over a Rayleigh fading channel.

## Objectives
- To analyze BER performance of MIMO systems
- To study the effect of Rayleigh fading on signal transmission

## Methodology
- MIMO system modeled using MATLAB
- Rayleigh fading channel applied
- BER vs SNR performance evaluated

## Tools Used
- MATLAB

## Domain
Communication Systems
